## Overview

The calendar system (`/calendar`) allows you to view fixer availability and book appointments directly. This ensures both you and the fixer have confirmed times for service delivery.

## Accessing the Calendar

You can access the calendar in two ways:

1. **From Job Offer Modal** - Click "Schedule Appointment" button
2. **Direct Navigation** - Go to `/calendar` page

### Setting Session Data

When scheduling from a job offer, the app stores necessary IDs:

```tsx Components/Job-offers/Job-offer-modal.tsx
const handleClickCalendar = () => {
  if (typeof window === 'undefined') return;
  const fixerId = offer.fixerId;
  const requesterId = sessionStorage.getItem('requester_id') ?? '';
  sessionStorage.setItem('fixer_id', fixerId);
  sessionStorage.setItem('requester_id', requesterId);
  sessionStorage.setItem('roluser', 'requester');
};
```

## View Modes

Switch between requester and fixer views to see different perspectives:

```tsx app/[locale]/calendar/page.tsx
const switchRole = () => {
  setUserRole((prevRole) => (prevRole === 'requester' ? 'fixer' : 'requester'));
};
```

## Appointment Context

The calendar uses React Context to manage appointment state:

```tsx app/[locale]/calendar/page.tsx
const providerValue = useMemo(
  () => ({
    isHourBookedFixer,
    isHourBooked,
    isEnabled,
    isCanceled,
    getAppointmentsForDay,
    refetchAll,
    refetchHour,
    loading,
  }),
  [
    isHourBookedFixer,
    isHourBooked,
    isEnabled,
    isCanceled,
    refetchAll,
    refetchHour,
    getAppointmentsForDay,
    loading,
  ],
);
```

This context provides:
- Real-time availability checks
- Booking status for specific hours
- Refresh functions to update data
- Loading states

## Troubleshooting

<AccordionGroup>
  <Accordion title="No Available Slots">
    If you don't see any available times:
    - Check different dates in the calendar
    - Contact the fixer directly via WhatsApp
    - Look for alternative fixers with similar services
  </Accordion>
  
  <Accordion title="Booking Fails">
    If booking doesn't complete:
    - Verify you're logged in as a requester
    - Check your internet connection
    - Refresh the page and try again
    - Ensure the time slot is still available
  </Accordion>
  
  <Accordion title="Calendar Not Loading">
    If calendar won't display:
    - Clear browser cache
    - Verify fixer_id is set in sessionStorage
    - Check console for error messages
    - Contact support if issue persists
  </Accordion>
</AccordionGroup>
