## Overview

After creating a job request, fixers respond with job offers containing their proposed pricing, timing, and service details. The Job Offers page helps you review and manage these proposals.

## Accessing Job Offers

Navigate to `/job-offer-list` to view all available job offers, including:

- Offers you've received from your requests
- General service offerings from fixers
- Promoted and featured services

## Taking Action on Offers

The modal footer provides several action buttons:

### Contact Options

<AccordionGroup>
  <Accordion title="WhatsApp Contact">
    Direct messaging with the fixer:
    
    ```tsx Components/Job-offers/Job-offer-modal.tsx
    const handleWhatsAppClick = () => {
      if (!whatsappClean) return;
      const url = `https://wa.me/${whatsappClean}`;
      window.open(url, '_blank');
    };
    ```
    
    - Opens WhatsApp chat in new window
    - Uses fixer's verified phone number
    - Enables quick communication
  </Accordion>
  
  <Accordion title="Schedule Appointment">
    Book directly on the fixer's calendar:
    
    ```tsx
    const handleClickCalendar = () => {
      const fixerId = offer.fixerId;
      const requesterId = sessionStorage.getItem('requester_id') ?? '';
      sessionStorage.setItem('fixer_id', fixerId);
      sessionStorage.setItem('requester_id', requesterId);
      sessionStorage.setItem('roluser', 'requester');
    };
    ```
    
    - Navigates to calendar view
    - Shows fixer's availability
    - Allows direct booking
  </Accordion>
  
  <Accordion title="Request Custom Job">
    Submit a specific job request to this fixer:
    
    - Opens job request modal
    - Pre-fills fixer information
    - Allows custom service description
    - Only available for logged-in requesters
    
    ```tsx
    const handleRequestJobClick = () => {
      if (!isAuthenticated) {
        router.push('/signUp');
        return;
      }
      if (user?.role === 'fixer') {
        return;
      }
      setIsMapModalOpen(true);
    };
    ```
  </Accordion>
  
  <Accordion title="Share Offer">
    Share the job offer with others:
    

    - Copies offer URL to clipboard

## Filtering & Sorting Offers

### Search Functionality

Use the search bar to find specific offers:

- Search by fixer name
- Search by service keywords
- Search by location
- Search by job title

```tsx app/[locale]/job-offer-list/page.tsx
const handleSearchSubmit = useCallback(
  (query: string) => {
    const trimmedQuery = query.trim();
    dispatch(setSearch(trimmedQuery));
    dispatch(resetPagination());
  },
  [dispatch],
);
```

### Advanced Filters

Open the filter drawer to refine results:

## Accepting an Offer

To accept an offer:

<Steps>
  <Step title="Review Details">
    Carefully read all offer terms, pricing, and conditions.
  </Step>
  
  <Step title="Contact Fixer">
    Use WhatsApp to discuss any questions or clarifications.
  </Step>
  
  <Step title="Schedule Appointment">

  </Step>
  
  <Step title="Confirm Details">

  </Step>

<Warning>

</Warning>

## Offer Status States

Offers can have different states:

```tsx app/components/fixers/RequestedJob.tsx
function getStateBadgeClasses(state: string) {
  switch (state) {
    case 'accepted':
      return 'bg-emerald-50 text-emerald-700 border-emerald-200';
    case 'pending':
      return 'bg-amber-50 text-amber-700 border-amber-200';
    case 'rejected':
      return 'bg-rose-50 text-rose-700 border-rose-200';
    case 'completed':
      return 'bg-blue-50 text-blue-700 border-blue-200';
    default:
      return 'bg-gray-50 text-gray-700 border-gray-200';
  }
}
```
