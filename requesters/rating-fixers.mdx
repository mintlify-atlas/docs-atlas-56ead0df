## Overview

After a job is completed, you can rate your experience and leave reviews to help other requesters make informed decisions and help fixers improve their services.

## Accessing Rated Jobs

Navigate to `/view-rate-jobs` to see all your completed jobs and their ratings.

### Rated Jobs List

The rated jobs page displays:

```tsx app/[locale]/view-rate-jobs/page.tsx
function RatedJobsList({ jobs, userId }: { jobs: RatedJob[]; userId?: string }) {
  return (
    <section className='space-y-6'>
      <div className='max-h-[65vh] overflow-y-auto'>
        {jobs.length === 0 ? (
          <p className='text-sm text-gray-500'>No hay trabajos calificados aún</p>
        ) : (
          <ul className='flex flex-col gap-4'>
            {jobs.map((job) => (
              <li key={job.id} className='flex items-center justify-between p-4 rounded-xl'>
                <div>
                  <p className='font-medium'>{job.title}</p>
                  <p className='text-xs text-gray-500'>
                    Date: {new Date(job.dateISO).toLocaleDateString()}
                  </p>
                </div>
                <StarRating value={job.rating ?? 0} />
              </li>
            ))}
          </ul>
        )}
      </div>
    </section>
  );
}
```

## Rating System

Servineo uses a 3-star rating system:

### Star Values

```tsx app/[locale]/view-rate-jobs/page.tsx
function StarRating({
  value,
  size = 38,
  srLabel,
}: {
  value: 0 | 1 | 2 | 3;
  size?: number;
  srLabel?: string;
}) {
  return (
    <div className='inline-flex gap-1' aria-label={srLabel ?? `${value} de 3 estrellas`}>
      {Array.from({ length: 3 }).map((_, i) => {
        const filled = i < value;
        return (
          <svg
            key={i}
            width={size}
            height={size}
            viewBox='0 0 24 24'
            className={filled ? 'drop-shadow-sm' : 'opacity-60'}
          >
            <path
              d='M12 17.27 18.18 21l-1.64-7.03L22 9.24l-7.19-.61L12 2 9.19 8.63 2 9.24l5.46 4.73L5.82 21z'
              style={
                filled
                  ? { fill: '#facc15' } // yellow
                  : { fill: 'transparent', stroke: '#cbd5e1', strokeWidth: 1.5 }
              }
            />
          </svg>
        );
      })}
    </div>
  );
}
```

## Rating a Completed Job

<Steps>
  <Step title="Job Completion">
    After the fixer marks the job as complete, you'll receive a notification to rate the service.
  </Step>
  
  <Step title="Access Rating Form">
    Click the notification or navigate to the completed job in your job list.
  </Step>
  
  <Step title="Select Star Rating">
    Choose 1, 2, or 3 stars based on your experience:
    - Click on the star corresponding to your rating
    - Stars are interactive and responsive
    - Selected stars appear filled in yellow
  </Step>
  
  <Step title="Write Review (Optional)">
    Add detailed feedback about your experience:
    - What went well
    - Areas for improvement
    - Specific skills demonstrated
    - Professionalism and communication
  </Step>
  
  <Step title="Submit Rating">
    Click submit to finalize your review.
    - Rating is immediately visible to other users
    - Fixer receives notification
    - Cannot be edited after submission
  </Step>
</Steps>

## Viewing Your Ratings

### Rated Jobs Page

The `/view-rate-jobs` page shows all your rating history:

```tsx app/[locale]/view-rate-jobs/page.tsx
export default function RatedJobsPage() {
  const { user } = useAppSelector((state) => state.user);
  const userId = user?._id || user?.id;
  
  const [jobs, setJobs] = useState<RatedJob[]>([]);
  const [currentSort, setCurrentSort] = useState<string>('recent');
  
  const fetchJobs = async (sortParam: string) => {
    const result = await apiFetch<{ data?: RatedJob[] }>(
      `api/rated-jobs/user/${userId}?sortBy=${sortParam}`,
    );
    setJobs(result.data ?? []);
  };
}
```

## Sorting Options

Organize your rated jobs using dropdown filters:

### Sort by Rating

```tsx app/[locale]/view-rate-jobs/page.tsx
const handleRatingSort = (key: string) => {
  setCurrentSort(key);
  fetchJobs(key);
};

<GenericDropdown
  label='Ordenar por calificación'
  onSelect={handleRatingSort}
  options={[
    { key: 'rating_desc', label: 'Descendente' },
    { key: 'rating_asc', label: 'Ascendente' },
  ]}
/>
```

- **Descending** - Highest rated jobs first (3 → 1 stars)
- **Ascending** - Lowest rated jobs first (1 → 3 stars)

### Sort by Date

```tsx
const handleDateSort = (key: string) => {
  setCurrentSort(key);
  fetchJobs(key);
};

<GenericDropdown
  label='Ordenar por fecha'
  onSelect={handleDateSort}
  options={[
    { key: 'recent', label: 'Mas reciente' },
    { key: 'oldest', label: 'Mas antiguo' },
  ]}
/>
```

- **Most Recent** - Newest ratings first
- **Oldest** - Earliest ratings first

## Writing Effective Reviews

<Steps>
  <Step title="Be Specific">
    Mention exact details about what was done well or poorly:
    - "Fixed the leaky faucet and replaced worn washers"
    - "Arrived 30 minutes late without notification"
  </Step>
  
  <Step title="Be Honest">
    Provide truthful feedback that helps others and the fixer:
    - Don't inflate ratings
    - Don't be overly harsh
    - Focus on facts
  </Step>
  
  <Step title="Be Constructive">
    If giving low ratings, explain how they could improve:
    - "Would be 3 stars with better communication"
    - "Great work but needs to improve punctuality"
  </Step>
  
  <Step title="Be Fair">
    Consider circumstances and context:
    - Weather delays
    - Material availability
    - Complexity beyond initial scope
  </Step>
</Steps>
