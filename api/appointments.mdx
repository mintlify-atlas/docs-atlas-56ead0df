## Overview

The Appointments API manages scheduling between requesters and fixers. This API handles appointment creation, updates, and status management.

<Info>
Appointments API endpoints are currently in development. This documentation will be updated as endpoints become available.
</Info>

## Planned Endpoints

### Create Appointment

Schedule a new appointment between a requester and a fixer.

```typescript
// Coming soon
const [createAppointment] = useCreateAppointmentMutation();

await createAppointment({
  requesterId: 'user-123',
  fixerId: 'fixer-456',
  jobOfferId: 'job-789',
  scheduledDate: '2026-03-20T10:00:00Z',
  duration: 60, // minutes
  location: '123 Main St',
}).unwrap();
```

<ParamField body="requesterId" type="string" required>
  ID of the user requesting the service
</ParamField>

<ParamField body="fixerId" type="string" required>
  ID of the fixer providing the service
</ParamField>

<ParamField body="jobOfferId" type="string" required>
  Related job offer ID
</ParamField>

<ParamField body="scheduledDate" type="string" required>
  ISO 8601 date-time string
</ParamField>

<ParamField body="duration" type="number" required>
  Appointment duration in minutes
</ParamField>

<ParamField body="location" type="string" required>
  Appointment location address
</ParamField>
