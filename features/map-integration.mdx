## Overview

Servineo integrates interactive maps powered by Leaflet and React-Leaflet to visualize job offers geographically. Users can see their location, nearby professionals, and filter by distance radius.


## Hover Interactions

Optimized hover system with debouncing:

```tsx title="Marker hover events" {1-2,4-17,19-29}
const markerElement = marker.getElement();
if (markerElement) {
  markerElement.addEventListener('mouseenter', () => {
    // Clear pending timeouts
    if (hoverTimeoutRef.current) {
      clearTimeout(hoverTimeoutRef.current);
      hoverTimeoutRef.current = null;
    }

    // Skip if already showing
    if (currentHoveredOfferIdRef.current === offer.id) {
      return;
    }

    pendingHoverRef.current = offer.id;

    // Short delay for responsiveness
    hoverTimeoutRef.current = setTimeout(() => {
      if (pendingHoverRef.current === offer.id && !isHoveringCardRef.current) {
        currentHoveredOfferIdRef.current = offer.id;
        setHoveredOffer(offer);
      }
      pendingHoverRef.current = null;
    }, 100);
  });

  markerElement.addEventListener('mouseleave', () => {
    if (pendingHoverRef.current === offer.id) {
      pendingHoverRef.current = null;
    }

    if (hoverTimeoutRef.current) {
      clearTimeout(hoverTimeoutRef.current);
      hoverTimeoutRef.current = null;
    }

    if (currentHoveredOfferIdRef.current === offer.id) {
      hoverTimeoutRef.current = setTimeout(() => {
        if (!isHoveringCardRef.current && 
            currentHoveredOfferIdRef.current === offer.id) {
          currentHoveredOfferIdRef.current = null;
          setHoveredOffer(null);
        }
      }, 150);
    }
  });
}
```

## Distance Calculation

Haversine formula for accurate distance calculation:

```tsx title="Calculate distance between coordinates" {1-2,3-11}
const calculateDistance = (
  lat1: number, lng1: number, 
  lat2: number, lng2: number
) => {
  const R = 6371; // Earth radius in km
  const dLat = ((lat2 - lat1) * Math.PI) / 180;
  const dLng = ((lng2 - lng1) * Math.PI) / 180;
  const a =
    Math.sin(dLat / 2) * Math.sin(dLat / 2) +
    Math.cos((lat1 * Math.PI) / 180) *
    Math.cos((lat2 * Math.PI) / 180) *
    Math.sin(dLng / 2) *
    Math.sin(dLng / 2);
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
  return R * c;
};
```

### Count Offers in Radius

```tsx title="Calculate offers within radius" {1-12}
const calculateOffersInRadius = useCallback(
  (currentRadius: number) => {
    const count = offers.filter((offer) => {
      const distance = calculateDistance(
        userLocation.lat,
        userLocation.lng,
        offer.location.lat,
        offer.location.lng,
      );
      return distance <= currentRadius;
    }).length;
    setOffersInRadius(count);
  },
  [offers],
);
```

## Performance Optimizations

1. **Lazy import Leaflet** - Only load on client side
2. **Memoized callbacks** - Prevent unnecessary re-renders
3. **Debounced hover** - Reduce state updates
4. **Staggered marker animations** - Smooth visual appearance
5. **Cleanup on unmount** - Prevent memory leaks

## Best Practices

1. **Always check client-side** before rendering Leaflet
2. **Clean up map instances** in useEffect cleanup
3. **Use refs for map state** to avoid re-renders
4. **Provide fallback UI** during loading
5. **Optimize marker icons** with CSS classes
6. **Debounce hover interactions** for better UX
7. **Show distance information** for context
