## Overview

Servineo implements a comprehensive notification system that supports multiple channels including in-app notifications, WhatsApp, email, and push notifications. The system uses the `nextjs-toast-notify` package for toast notifications and includes a custom notification center with filtering, pagination, and offline support.

### Notification Categories

1. **WhatsApp Notifications** (`whatsapp`): Messages sent via WhatsApp
2. **Email Notifications** (`email`): Email alerts and updates
3. **Appointment Notifications** (`citas`): Booking and appointment reminders
4. **Offers/Promotions** (`ofertas_promociones`): Special offers and promotional messages
5. **Wallet Notifications** (`billetera`): Payment and balance updates
6. **Unread Filter** (`no_leidas`): Show only unread notifications

## Time Ago Helper

Utility function for human-readable timestamps:

```typescript
function timeAgo(date: string | Date): string {
  const now = new Date();
  const past = new Date(date);
  const diffMs = now.getTime() - past.getTime();
  const diffMins = Math.floor(diffMs / 60000);
  const diffHours = Math.floor(diffMs / 3600000);
  const diffDays = Math.floor(diffMs / 86400000);

  if (diffMins < 1) return 'Ahora';
  if (diffMins < 60) return `Hace ${diffMins} min`;
  if (diffHours < 24) return `Hace ${diffHours} h`;
  return `Hace ${diffDays} días`;
}
```

## Best Practices

1. **Always show loading states** during notification fetch
2. **Implement offline caching** for better UX
3. **Use optimistic updates** for read status
4. **Provide clear visual feedback** for different notification types
5. **Enable filtering** for easier navigation
6. **Implement pagination** for large notification lists
7. **Support deep linking** via action_url
8. **Auto-close success notifications** but require user action for warnings
9. **Mask sensitive data** in notifications
10. **Retry failed operations** when connection is restored
11. **Use appropriate icons** for different notification types
12. **Show relative timestamps** ("2 hours ago") instead of absolute dates
13. **Implement sound/vibration** for important notifications (mobile)
14. **Group similar notifications** to reduce clutter
15. **Allow bulk operations** (mark all as read, clear all)
