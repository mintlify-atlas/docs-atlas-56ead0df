The Servineo statistics module provides comprehensive analytics for tracking user searches, fixer distributions, job types, and geographic data across different time periods.

## Overview

The statistics dashboard is accessible at `/adminStatistic` and provides real-time analytics with interactive charts and filtering capabilities.

## Statistics API


## Time Period Filtering

### Available Periods

The statistics can be viewed across three time periods:

- **Weekly (semanal)**: `/searches/semanal`
- **Monthly (mensual)**: `/searches/month`
- **Annual (anual)**: `/searches/year`

### Period Selection

```typescript
const [selectedPeriod, setSelectedPeriod] = useState<'semanal' | 'mensual' | 'anual'>('mensual');

<select
  value={selectedPeriod}
  onChange={(e) => setSelectedPeriod(e.target.value as 'semanal' | 'mensual' | 'anual')}
>
  <option value='semanal'>{t('periods.weekly')}</option>
  <option value='mensual'>{t('periods.monthly')}</option>
  <option value='anual'>{t('periods.yearly')}</option>
</select>
```

### Auto-Refetch on Period Change

```typescript

  refetch();
}, [selectedPeriod, refetch]);
```

#### Donut Chart Implementation

```tsx
export function ChartPieDonut({ data, title, description }: ChartPieDonutProps) {
  const dataWithColors = data.map((item, index) => ({
    ...item,
    fill: item.fill || ['#FFBB28', '#EF4444', '#60A5FA'][index % 3],
  }));

  return (
    <div className='flex flex-col border rounded-lg bg-white shadow-sm p-6'>
      <PieChart width={250} height={250}>
        <Tooltip />
        <Pie
          data={dataWithColors}
          dataKey='value'
          nameKey='name'
          innerRadius={60}
          outerRadius={100}
          paddingAngle={2}
        />
      </PieChart>
    </div>
  );
}
```

## Data Transformation

The statistics data is transformed from the API format to chart-compatible format:

```typescript
const transformFilterData = (filterData: Record<string, number> | undefined) => {
  if (!filterData) return [];
  return Object.entries(filterData).map(([name, value]) => ({
    name,
    value: value as number,
  }));
};
```

## Job Activity Logs

A separate statistics API tracks detailed job activity:

```typescript
export interface JobLog {
  _id: string;
  userId: string;
  date: string;
  role: 'visitor' | 'requester' | 'fixer';
  type:
    | 'login'
    | 'search'
    | 'click'
    | 'review'
    | 'session_start'
    | 'session_end'
    | 'daily_jobs_status';
  metadata: JobStatusMetadata;
  timestamp: string;
}

export interface JobStatusMetadata {
  completed: number;
  pending: number;
  inProgress: number;
  total: number;
  collectionDate: string;
}
```
