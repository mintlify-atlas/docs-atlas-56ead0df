## Overview

The Advanced Search feature provides power users with granular control over search parameters. It combines all basic filters with additional options like price ranges, date filtering, rating selection, and tag-based filtering.

## Search Options

### Search Modifiers

The `SearchCheckboxes` component provides two key search modifiers:

```tsx title="src/Components/Adv-search/SearchCheckboxes.tsx" {13-18}
export function SearchCheckboxes({
  titleOnly,
  setTitleOnly,
  exactWords,
  setExactWords,
}: SearchCheckboxesProps) {
  const t = useTranslations('search');

  return (
    <div className="w-full bg-gray-100 border rounded-lg p-4 mt-4">
      <div className="flex flex-col gap-3">
        <label className="flex items-center gap-3">
          <input
            type="checkbox"
            checked={titleOnly}
            onChange={(e) => {
              const checked = e.target.checked;
              setTitleOnly(checked);
              if (checked) setExactWords(false);
            }}
          />
          <span>{t('titleOnly')}</span>
        </label>

        <label className="flex items-center gap-3">
          <input
            type="checkbox"
            checked={exactWords}
            onChange={(e) => {
              const checked = e.target.checked;
              setExactWords(checked);
              if (checked) setTitleOnly(false);
            }}
          />
          <span>{t('exactWords')}</span>
        </label>
      </div>
    </div>
  );
}
```

**Search Modes:**

- **Title Only**: Searches only in job offer titles
- **Exact Words**: Matches exact phrases in title and description
- Mutually exclusive options

### 4. Price Ranges

Dynamic price filtering with API-backed ranges:

```tsx title="src/Components/Adv-search/PriceRangeList.tsx" {23-29}
const PriceRangeList: React.FC<PriceRangeListProps> = ({ 
  onFilterChange, 
  clearSignal 
}) => {
  const t = useTranslations('advancedSearch');
  const [selectedRanges, setSelectedRanges] = useState<string[]>([]);
  const [ranges, setRanges] = useState<RangeItem[]>([]);

  // RTK Query for price ranges
  const { 
    data: priceRangesData, 
    isLoading, 
    error: queryError 
  } = useGetPriceRangesQuery();

  // Handle checkbox changes
  const handleCheckboxChange = (rangeValue: string) => {
    const newSelectedRanges = selectedRanges.includes(rangeValue)
      ? selectedRanges.filter((range) => range !== rangeValue)
      : [...selectedRanges, rangeValue];

    setSelectedRanges(newSelectedRanges);
    const priceKey = newSelectedRanges.length === 1 
      ? newSelectedRanges[0] 
      : '';
    onFilterChange?.({ priceRanges: newSelectedRanges, priceKey });
  };
  // ...
}
```

**Features:**

- Fetches price ranges from API using RTK Query
- Caches ranges in sessionStorage for performance
- Formats currency labels (e.g., "$90" → "90bs")
- Supports single or multiple range selection

## Clear Filters

Users can reset all filters to default state:

```tsx title="Clear all filters" {2-14}
<ClearButton
  onClick={() => {
    setSearchQuery('');
    setSelectedRanges([]);
    setSelectedCity([]);
    setSelectedJobs([]);
    setSelectedCategories([]);
    setSelectedPriceRanges([]);
    setTitleOnly(false);
    setExactWords(false);
    setResultsCount(null);
    setSelectedDateFilter('specific');
    setSelectedSpecificDate(null);
    setSelectedRating(null);
    setClearSignal((s) => s + 1);
    fetchGlobalTotal();
  }}
/>
```

## Keyboard Shortcuts

Advanced search supports keyboard navigation:

- **ESC**: Exit advanced search and return to job offers list

```tsx title="ESC key handler"
useEffect(() => {
  const onKeyDown = (e: KeyboardEvent) => {
    if (e.key === 'Escape') {
      router.push('/job-offer-list');
    }
  };

  window.addEventListener('keydown', onKeyDown);
  return () => window.removeEventListener('keydown', onKeyDown);
}, [router]);
```

## State Persistence

Advanced search state is preserved in sessionStorage:

```tsx
// Save state
window.sessionStorage.setItem('advSearch_state', JSON.stringify(state));

// Clear state on reset
window.sessionStorage.removeItem('advSearch_state');
window.sessionStorage.removeItem('fromAdv');
window.sessionStorage.removeItem('appliedFilters');
```

## Performance Optimizations

1. **Price range caching** in sessionStorage
2. **Debounced URL updates** to prevent excessive history entries
3. **Conditional rendering** of filter sections (only open sections render content)
4. **Memoized filter lists** to prevent recalculation
5. **Lazy loading** of dropdown options

## Best Practices

1. **Provide real-time feedback** on results count
2. **Preserve user selections** across sessions when appropriate
3. **Support keyboard navigation** for power users
4. **Make filters mutually exclusive** when it makes sense (titleOnly vs exactWords)
5. **Show clear visual indicators** for active filters
6. **Enable URL sharing** for collaborative searching
7. **Offer quick reset** to clear all filters
