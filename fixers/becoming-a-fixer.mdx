## Registration Process

Becoming a Fixer on Servineo involves completing a multi-step registration wizard that collects your professional information and verifies your credentials.

## Step 1: Initial Registration

The registration form collects your basic contact information:

```typescript
interface InitialRegistrationData {
  name: string;      // Full name (max 30 characters)
  email: string;     // Valid email address
  phone: string;     // Contact phone number
}
```

### Implementation

The initial form uses React Hook Form with Zod validation:

```tsx
import { useForm, Controller } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { initialRegistrationSchema } from '@/app/lib/validations/fixer-schemas';

const { control, handleSubmit, formState: { errors } } = useForm<InitialRegistrationData>({
  resolver: zodResolver(initialRegistrationSchema),
  defaultValues: {
    name: '',
    email: '',
    phone: '',
  },
  mode: 'onChange',
});
```

## Submission

The wizard submits data to the backend using RTK Query:

```typescript
const [convertToFixer] = useConvertToFixerMutation();

const onSubmit: SubmitHandler<FixerProfileData> = async (data) => {
  const payload = {
    id: user._id?.toString(),
    profile: {
      telefono: user.telefono,
      ci: data.ci.trim(),
      services: data.servicios.map(id => {
        const service = DEFAULT_SERVICES.find(s => s.id === id);
        return { name: service?.name ?? 'Servicio desconocido' };
      }),
      vehicle: data.vehiculo,
      paymentMethods: [
        data.metodoPago.hasEfectivo && { type: 'efectivo' },
        data.metodoPago.qr && { type: 'qr' },
        data.metodoPago.tarjetaCredito && { type: 'tarjeta' },
      ].filter(Boolean),
      location: {
        lat: Number(data.workLocation.lat),
        lng: Number(data.workLocation.lng),
        direccion: data.workLocation.direccion?.trim() || '',
      },
      terms: { accepted: data.acceptTerms },
    },
  };

  await convertToFixer(payload).unwrap();
};
```

## Profile Status

After registration, your profile enters a "pending" status for review:

```typescript
interface IUser {
  fixerProfile: 'pending' | 'completed' | 'rejected';
}
```

Once approved, you can access the full fixer dashboard and start creating job offers.
