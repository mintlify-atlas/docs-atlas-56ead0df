## Reputation System Overview

Your reputation on Servineo is built through ratings, reviews, and your job completion record. A strong reputation leads to:

- More client bookings
- Higher visibility in search results
- Ability to charge premium rates
- Increased trust from new clients

## Rating System

Fixers receive star ratings from 0 to 5:

```typescript
interface Fixer {
  rating?: number;        // Average rating (0-5)
  completedJobs: number;  // Total jobs completed

```

## Job Statistics

Your performance is tracked across multiple metrics:

### Completion Rate

The ratio of completed to total jobs:

```typescript
interface JobStats {
  completedJobs: number;
  cancelledJobs: number;
  totalJobs: number;
  completionRate: number;  // Percentage
}

const calculateCompletionRate = (completed: number, cancelled: number) => {
  const total = completed + cancelled;
  return total > 0 ? (completed / total) * 100 : 0;
};
```

### Monthly Performance

Track job trends over time:

```typescript
interface MonthlyData {
  month: string;
  completados: number;  // Completed jobs
  cancelados: number;   // Cancelled jobs
}

interface Fixer {
  monthlyData?: MonthlyData[];
}
```

Display monthly statistics:

```tsx

<FixerGraficCard
  completedJobs={fixer.completedJobs ?? 0}
  cancelledJobs={fixer.cancelledJobs ?? 0}
  monthlyData={fixer.monthlyData}
/>
```

## Join Date Display

Your tenure on the platform builds trust:

```tsx
const formattedJoinDate = (() => {
  if (!fixer.joinDate) return null;
  const date = new Date(fixer.joinDate);
  if (Number.isNaN(date.getTime())) return null;

  return date.toLocaleDateString('es-ES', {
    month: 'long',
    year: 'numeric',
  });
})();

{formattedJoinDate && (
  <p className='mt-1 text-sm opacity-80'>
    En la app desde {formattedJoinDate}
  </p>
)}
```
